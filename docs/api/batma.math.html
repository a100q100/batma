batma.math — Math Helpers¶

Constants¶

batma.math.e¶

Represents the mathematical constant e.

batma.math.log10e¶

Represents the log base ten of e.

batma.math.log2e¶

Represents the log base two of e.

batma.math.pi¶

Represents the value of pi.

batma.math.pi_over_2¶

Represents the value of pi divided by two.

batma.math.pi_over_4¶

Represents the value of pi divided by four.

batma.math.two_pi¶

Represents the value of pi times two.

Functions¶

batma.math.clamp(value, min_value, max_value)¶

Restricts a value to be within a specified range. Returns min_value if x < min_value, max_value if x > max_value, otherwise x.

batma.math.distance(value1, value2)¶

Calculates the absolute value of the difference of two values.

batma.math.inversesqrt(value)¶

Returns 1/sqrt(value).

batma.math.mod(a, b)¶

Returns a%b. This is just an equivalent for the %-operator.

batma.math.sign(value)¶

Returns -1 with a negative argument, +1 with a positive argument, and 0 if its argument is zero.

batma.math.step(value, threshold)¶

Returns 0 if value < threshold, otherwise 1.

batma.math.wrap_angle(angle)¶

Reduces a given angle to a value between π and -π.

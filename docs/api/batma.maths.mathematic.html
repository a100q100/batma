batma.math — Math Helpers¶

Constants¶

batma.maths.mathematic.e¶

Represents the mathematical constant e.

batma.maths.mathematic.log10e¶

Represents the log base ten of e.

batma.maths.mathematic.log2e¶

Represents the log base two of e.

batma.maths.mathematic.pi¶

Represents the value of pi.

batma.maths.mathematic.pi_over_2¶

Represents the value of pi divided by two.

batma.maths.mathematic.pi_over_4¶

Represents the value of pi divided by four.

batma.maths.mathematic.two_pi¶

Represents the value of pi times two.

Functions¶

batma.maths.mathematic.catmullrom(value1, value2, value3, value4, amount)[source]¶

Performs a Catmull-Rom interpolation using the specified positions.

batma.maths.mathematic.clamp(value, min_value, max_value)[source]¶

Restricts a value to be within a specified range. Returns min_value if x < min_value, max_value if x > max_value, otherwise x.

batma.maths.mathematic.distance(value1, value2)[source]¶

Calculates the absolute value of the difference of two values.

batma.maths.mathematic.hermite(value1, tangent1, value2, tangent2, amount)[source]¶

Performs a Hermite spline interpolation.

batma.maths.mathematic.inversesqrt(value)¶

Returns 1/sqrt(value).

batma.maths.mathematic.lerp(value1, value2, amount)[source]¶

This method performs the linear interpolation based on the following formula.

value1 + (value2 - value1) * amount

Passing amount a value of 0 will cause value1 to be returned, a value of 1 will cause value2 to be returned.

batma.maths.mathematic.mod(a, b)¶

Returns a%b. This is just an equivalent for the %-operator.

batma.maths.mathematic.sign(value)¶

Returns -1 with a negative argument, +1 with a positive argument, and 0 if its argument is zero.

batma.maths.mathematic.smoothstep(value1, value2, amount)[source]¶

Interpolates between two values using a cubic equation.

batma.maths.mathematic.step(value, threshold)¶

Returns 0 if value < threshold, otherwise 1.

batma.maths.mathematic.wrap_angle(angle)[source]¶

Reduces a given angle to a value between π and -π.

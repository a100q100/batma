Release 0.1 (in development)¶

Features added¶

  • Input state for keyboard and mouse;
  • Sprites and Texts with easy rotation, scaling, positioning;
  • 2D and 3D math library;
  • Scheduling of user defined functions;
  • Interpolation functions;
  • Iteration functions;
  • Scene based;
